# Food Delivery App

A full-stack food delivery website built using:

- React.js
- Node.js
- Express.js
- MongoDB Atlas
- Stripe Payment Gateway

## Features

- User Authentication
- Add to Cart
- Place Orders
- Stripe Payments
- Order Tracking
- Admin Dashboard
- Food Management

## Tech Stack

Frontend: React, CSS, Axios
Backend: Node.js, Express.js
Database: MongoDB Atlas
Payment: Stripe
